# essence

用代码主动触碰真实世界的本质。

不是学习笔记，不是算法练习。每个模块都是对某个底层规律的最小实现——小到足以让人感受到它，不追求完整。

## 模块

| 模块 | 本质 | 最小目标 |
|------|------|----------|
| `light/` | 光的传播 | 一条光线，一个球，一个像素 |
| `motion/` | 力与运动 | 重力下的自由落体，只用牛顿第二定律 |
| `wave/` | 频率与分解 | 手写 DFT，把信号拆成频率成分 |
| `rhythm/` | 数与对齐 | 两个节奏等待同时落下的那一刻 |

## 运行

```bash
cargo run -p light
cargo run -p motion
cargo run -p wave
cargo run -p rhythm
```
